# 请假管理规则

## 概述

本文档详细说明UNISSA人力资源系统中的请假管理规则，包括请假类型、余额计算、审批流程等。

## 请假类型

系统支持以下请假类型：

| 请假类型 | 英文标识 | 说明 | 余额校验 |
|---------|---------|------|---------|
| 年假 | annual | 员工年度休假 | 需校验余额 |
| 病假 | medical | 因病请假 | 需校验余额 |
| 无薪假 | unpaid | 无薪资的请假 | 不校验余额 |
| 产假 | maternity | 女性员工生育假 | 需校验余额 |
| 陪产假 | paternity | 男性员工陪产假 | 需校验余额 |
| 紧急假 | emergency | 紧急情况请假 | 需校验余额 |

## 请假天数计算

请假天数计算规则：
- 开始日期和结束日期都包含在请假范围内
- 计算公式：`请假天数 = (结束日期 - 开始日期) ÷ 86400000 + 1`
- 最小请假天数为1天
- 结束日期必须晚于开始日期

## 余额校验规则

### 1. 年假和病假
- 系统会检查员工的剩余假期余额
- 如果请求的请假天数超过可用余额，申请将被拒绝
- 错误提示：`Insufficient leave balance. Available: {可用天数} days, Requested: {请求天数} days`

### 2. 无薪假
- 无薪假不校验余额
- 员工可以申请任意天数的无薪假

## 审批流程

### 1. 审批级别
- **一级审批**：适用于请假天数 < 3天的申请
- **二级审批**：适用于请假天数 ≥ 3天的申请

### 2. 审批流程

#### 短假（< 3天）
1. 员工提交请假申请
2. 状态：`pending`（待处理）
3. 审批人（manager/admin/hradmin）审批
4. 状态：`approved`（已批准）或 `rejected`（已拒绝）

#### 长假（≥ 3天）
1. 员工提交请假申请
2. 状态：`pending_hr`（等待HR审批）
3. HR审批通过后，状态变为 `pending_manager`（等待经理审批）
4. 经理审批后，状态变为 `approved`（已批准）或 `rejected`（已拒绝）

### 3. 审批权限
- `manager`：可审批所有请假申请
- `admin`：可审批所有请假申请
- `hradmin`：可审批所有请假申请
- 普通员工：只能查看自己的请假记录

## 请假状态

| 状态 | 描述 | 颜色标识 |
|------|------|---------|
| pending | 待处理（短假） | 橙色 |
| pending_hr | 等待HR审批 | 蓝色 |
| pending_manager | 等待经理审批 | 紫色 |
| approved | 已批准 | 绿色 |
| rejected | 已拒绝 | 红色 |
| cancelled | 已取消 | 灰色 |

## 代理人设置

- 员工在提交请假申请时可以指定代理人
- 代理人负责在员工请假期间处理其职责
- 代理人信息会存储在请假记录中

## 假期余额扣减

- 只有请假申请被**最终批准**后，才会扣减相应的假期余额
- 扣减规则：
  - 年假：扣减 `leaveBalanceAnnual`
  - 病假：扣减 `leaveBalanceMedical`
  - 无薪假：不扣减余额

## 系统操作

### 提交请假申请

**API**：`POST /api/v1/hr/leave`

**请求参数**：
```json
{
  "leaveType": "annual",
  "startDate": "2026-04-10",
  "endDate": "2026-04-13",
  "reason": "家庭旅行",
  "coveringOfficerId": "STF-002"
}
```

**响应**：
- 成功：`{"success": true, "data": {...}, "message": "Leave request submitted successfully"}`
- 失败：`{"success": false, "message": "Insufficient leave balance. Available: 3 days, Requested: 4 days"}`

### 审批请假申请

**API**：`PATCH /api/v1/hr/leave/:id/approve`

**请求参数**：
```json
{
  "action": "approved",
  "remarks": "批准请假"
}
```

## 常见问题

### Q: 为什么我的请假申请被拒绝？
**A**: 可能的原因：
1. 假期余额不足
2. 审批人拒绝了申请
3. 提交的信息不完整或有误

### Q: 如何查看我的请假余额？
**A**: 员工可以在个人门户页面查看自己的假期余额

### Q: 代理人可以查看被代理人的请假记录吗？
**A**: 目前系统不支持代理人查看被代理人的请假记录

### Q: 请假申请可以取消吗？
**A**: 目前系统暂不支持取消已提交的请假申请
