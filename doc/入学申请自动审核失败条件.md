# 入学申请自动审核失败条件

## 概述

本文档详细说明UNISSA学生管理系统中入学申请自动审核失败的具体条件。系统会根据申请人提交的学历信息进行自动审核，不符合条件的申请将被标记为自动审核失败。

## 自动审核失败的条件

### 1. 学历资格不被认可

**条件**：申请人的最高学历不在系统认可的学历列表中。

**认可的学历类型**：
- O_LEVEL
- A_LEVEL
- DIPLOMA
- DEGREE
- MASTERS

**注意**：系统支持大小写不敏感匹配，例如 `a_level`、`diploma` 等也会被认可。

**代码实现**：
```typescript
const qualOrder = ['O_LEVEL', 'A_LEVEL', 'DIPLOMA', 'DEGREE', 'MASTERS',
                   'o_level', 'a_level', 'diploma', 'degree', 'masters']
const qualIdx = qualOrder.indexOf(String(highestQualification))
const autoCheckPassed = qualIdx >= 0 && (!yearOfCompletion || (currentYear - completionYear) <= 10)
```

### 2. 学历完成时间超过10年

**条件**：申请人填写了学历完成年份，且该年份距今超过10年。

**计算方式**：
- 系统获取当前年份：`const currentYear = new Date().getFullYear()`
- 计算毕业年限：`currentYear - completionYear`
- 如果毕业年限 > 10，则自动审核失败

**代码实现**：
```typescript
const completionYear = Number(yearOfCompletion)
const currentYear = new Date().getFullYear()
const autoCheckPassed = qualIdx >= 0 && (!yearOfCompletion || (currentYear - completionYear) <= 10)
```

## 自动审核结果

| 条件组合 | 审核结果 | 状态 |
|---------|---------|------|
| 学历类型无效 | 自动失败 | `auto_check_failed` |
| 学历类型有效 + 毕业超过10年 | 自动失败 | `auto_check_failed` |
| 学历类型有效 + 未填写毕业年份 | 通过初筛 | `under_review` |
| 学历类型有效 + 毕业10年内 | 通过初筛 | `under_review` |

## 系统处理流程

1. **申请提交**：申请人填写并提交入学申请
2. **自动审核**：系统根据上述条件进行自动审核
3. **状态更新**：
   - 符合条件：状态设置为 `under_review`，进入人工审核流程
   - 不符合条件：状态设置为 `auto_check_failed`
4. **通知申请人**：系统自动通知申请人审核结果

## 申请人操作指南

如果申请被标记为自动审核失败，申请人可以：

1. **修改学历信息**：更新学历类型或毕业年份
2. **重新提交**：修改后重新提交申请，系统会重新进行自动审核
3. **联系招生办**：如果对审核结果有异议，可以联系招生办公室

### 核心代码位置

**文件**：`backend/src/routes/admissions.ts`

**关键代码**：
```typescript
// 自动资格检查
const qualOrder = ['O_LEVEL', 'A_LEVEL', 'DIPLOMA', 'DEGREE', 'MASTERS',
                   'o_level', 'a_level', 'diploma', 'degree', 'masters']
const qualIdx = qualOrder.indexOf(String(highestQualification))
const completionYear = Number(yearOfCompletion)
const currentYear = new Date().getFullYear()
const autoCheckPassed = qualIdx >= 0 && (!yearOfCompletion || (currentYear - completionYear) <= 10)
const autoStatus = autoCheckPassed ? 'under_review' : 'auto_check_failed'
```

### 状态管理

系统使用以下状态来管理申请流程：
- `auto_check_failed`：自动审核失败
- `under_review`：等待人工审核
- `approved`：申请通过
- `rejected`：申请被拒绝

## 常见问题

### Q: 为什么我的申请被自动审核失败？
**A**: 可能的原因：
1. 您填写的学历类型不在系统认可的列表中
2. 您的学历完成时间超过了10年

### Q: 如何修改我的申请？
**A**: 您可以登录系统，找到您的申请记录，点击"修改"按钮，更新学历信息后重新提交。

### Q: 如果我没有毕业年份怎么办？
**A**: 如果您不填写毕业年份，系统会默认通过初筛，进入人工审核流程。

## 总结

自动审核失败的条件主要有两个：
1. **学历类型不被认可**：必须是系统认可的学历类型之一
2. **毕业年限超过10年**：如果填写了毕业年份，必须在10年内

系统设计的目的是确保申请人的学历信息符合基本要求，同时为特殊情况保留人工审核的通道。
