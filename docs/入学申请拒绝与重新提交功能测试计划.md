# 入学申请拒绝与重新提交功能测试计划

## 测试目标
验证当入学申请被管理员拒绝后，申请人能够查看具体的拒绝原因，对申请内容进行修改和完善，然后重新提交申请。系统需支持申请状态从"已拒绝"变更为"待审核"，并保留修改历史记录以便追溯。

## 测试环境
- 前端: http://localhost:5174
- 后端: http://localhost:4000
- 测试用户: zara (学生角色)

### 数据库模型

```prisma
model ApplicantHistory {
  id              String   @id @default(cuid())
  applicantId     String
  applicant       Applicant @relation(fields: [applicantId], references: [id], onDelete: Cascade)
  previousStatus  String
  newStatus       String
  reason          String?
  changedBy       String?
  changedAt       DateTime @default(now())

  @@map("applicant_history")
}

model Applicant {
  // ... 其他字段
  histories       ApplicantHistory[]
}
```
